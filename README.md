# Health Clinic Queue Manager

This is a Flask web application that manages a clinic patient queue.

Features
- Register patients
- View waiting list
- Call next patient
- Track total patients seen

Technologies
- Python
- Flask
- OOP
- Queue Data Structure
- datetime module

How to Run

1 Install Flask

pip install flask

2 Run the application

python app.py

3 Open browser

http://127.0.0.1:5000
